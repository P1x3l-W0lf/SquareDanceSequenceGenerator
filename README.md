# SquareDanceSequenceGenerator
A python script to synthesize large amounts of input data and output square dance call sequences.

### ✨ Introduction
This project is a terminal-based python script that generates square dance call sequences of different difficulties based on input provided by a relative of mine and saves them to files on the system. It won't really work for anyone else without edits to the file path variables.

### ✨ Features
• create multiple sequences in one run

• Save them to a file named whatever you would like

• Let's you know if the file name is already in use to prevent data loss

• Sequences ranked by difficulty (easy, medium, hard)

### ▶️ Usage
Run the script.py file

Input the number of sequences you would like generated

Input the difficulty level of the sequences you want

The result will be displayed in the terminal

Input the name of the file you would like the sequences saved to

The program will make the file and save the sequences automatically.

### 📦 Dependencies
This project:

• Uses Python and requires that Python be installed on the device

• Requires specific file names and paths to function properly

### ⚙️ Configuration
If you really want to run the project you will need to:

• Have zip files of the data you would like to use labeled easy.zip, medium.zip, and hard.zip

• Update the data_folder variable to be the correct file path to the zip files on your device

### 🛠️ Troubleshooting
If you have any issues running this project, make sure:

• Python is installed

• The data files are all there and named correctly

• The file path in the data_folder variable is correct

• Your device has enough space to save the output to a file
